# ✨ 富文本编辑器 - 强大、轻量、易集成

🚀 一款现代化的开源富文本编辑器，专为开发者打造，提供流畅的编辑体验和丰富的功能扩展

## ⚡ 快速开始

只需一行代码，轻松集成到您的项目中！

```js
const content = `<p>开始您的富文本创作之旅</p>`

// 打开编辑器窗口，返回编辑后的内容
const editedContent = await import('https://huodoushigemi.github.io/online-doc/function.js').then(e => e.openDoc({ content }))
```
